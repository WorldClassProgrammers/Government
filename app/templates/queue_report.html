QueueReport

This API is to report queue (date and time) for a vaccine reservation

Here is the Rule:

[POST]

  • All 2 form data parameters are required! [citizen_id, queue]
  • citizen_id: need to be number of 13 digits
  • citizen_id: must have been registered
  • queue: need to be in format ['%Y-%m-%d %H:%M:%S.%f']
