Registration

This API is to register citizen information

Here is the Rule:

[GET] /registration/[citizen_id] - Return a registration (citizen) data as json

  • Sample Output:

    • {
      "citizen_id": "1234567890321",
      "name": "มารุณี",
      "surname": "ทองแท่ง",
      "birth_date": "1999-11-01",
      "occupation":"นิสิต",
      "phone_number": "0900900990",
      "is_risk": "False",
      "address": "สักที่ในโลก 199-e-3",
      "vaccine_taken": "['Astra']"
    }

[POST] /registration - To register citizen information

  • All 8 form data parameters are required! [citizen_id, name, surname, birth_date, occupation, phone_number, is_risk, address]
  • citizen_id: must be unique
  • citizen_id: need to be number of 13 digits
  • birth_date: need to be in one of available formats ['%d %b %Y', '%d-%m-%Y', '%Y-%m-%d', '%d/%m/%Y', '%Y/%m/%d']
  • birth_date: citizen must be at least 12 years old

[DELETE] /registration/[citizen_id] - Delete this citizen data and all of the reservations

  • Error only show for admin (logging)
  • Redirect to Citizen Database
