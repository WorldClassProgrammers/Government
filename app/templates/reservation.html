[POST] /reservation - To reserve vaccine reservation

  • All 3 form data parameters are required! [citizen_id, site_name, vaccine_name]
  • citizen_id: need to be number of 13 digits
  • citizen_id: must have been registered
  • vaccine_name: need to be one of available vaccine [Pfizer, Astra, Sinofarm, or Sinovac]
  • vaccine_name: need to follow vaccination orders

[DELETE] /reservation/[citizen_id] - To cancel the citizen unchecked vaccine reservation

  • Only 1 key is required! [citizen_id]
  • citizen_id: need to be number of 13 digits
  • citizen_id: must be of the one that has a reservation
